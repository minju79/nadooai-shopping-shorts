    상품 키워드 입력

  4. 4
    내 쇼츠 제작

    직접 촬영, 허가된 이미지, 상품 상세 이미지 중심으로 대본과 자막을 만듭니다.

영상 다운로드 도구

참고할 영상 링크를 넣고 다운로드 도구를 엽니다. 본인 영상이나 사용 허가가 있는 자료에만 사용하세요.

링크 복사 후 열기

다운로드 사이트가 열리면 방금 복사된 링크를 붙여넣으세요. 플랫폼 약관과 저작권은 직접 확인해야 합니다.

자막 추출

다운받은 영상 파일에서 음성을 인식해 TXT, SRT, VTT 자막을 만듭니다.

로컬 Whisper

처음 실행할 때는 Whisper 모델을 내려받느라 시간이 걸릴 수 있습니다.

영상 원본의 저작권과 플랫폼 약관을 확인한 뒤 사용하세요. 긴 영상은 처리 시간이 길어질 수 있습니다.

게시 업로드 허브

완성된 쇼츠 MP4와 캡션을 준비한 뒤, 각 플랫폼 업로드 화면으로 바로 이동합니다.

직접 업로드 모드

MP4 파일을 선택하면 공유 지원 여부를 확인합니다.

플랫폼별 캡션 초안 영상 파일명과 상품 키워드로 틱톡, 인스타, 쓰레드, 유튜브용 문구를 만듭니다.
업로드 화면 열기

틱톡은 일반 업로드, 틱톡 스튜디오는 광고/크리에이티브 제작 도구입니다. 용도가 달라서 둘 다 기본값으로 둡니다.

인스타와 유튜브는 업로드 화면에서 파일을 직접 선택해야 합니다. 쓰레드는 캡션을 URL에 실어 작성창을 열 수 있습니다.

자동 게시 API는 별도 로그인, 권한 승인, 앱 심사, 비즈니스/크리에이터 계정 조건이 필요합니다.
